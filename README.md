## About
Your Little Oyachi is a cozy little simulator where you take care of “Little Oyachi”, a tiny, extra-cute version of my girlfriend, Oyachi. Pet her, hang out with her, and keep her happy while you spend time together in this tiny world hehe.

This game started as an anniversary plan, then turned into a Valentine’s gift… but I ended up finishing it and decided to release it as a surprise instead.

> [!IMPORTANT]
> this version is NOT getting updates anymore, a new game is being planned with more features, more interactions, and more fun stuff soon!
